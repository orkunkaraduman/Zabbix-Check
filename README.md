# NAME

Zabbix::Check - Zabbix checks

# VERSION

version 1.01

# SYNOPSIS

Zabbix checks

# INSTALLATION

To install this module type the following

        perl Makefile.PL
        make
        make test
        make install

from CPAN

        cpan -i Zabbix::Check
